# TUK-ConvoSearch

An AI-powered student assistant for Technical University of Kenya using Retrieval-Augmented Generation (RAG).

## Features
- Answer student questions using official university documents
- Source citations for transparency
- 24/7 availability
- Web interface and API

## Tech Stack
- Python, FastAPI (Backend)
- React (Frontend)
- ChromaDB (Vector Database)
- Ollama + tinyllama (AI Model)
